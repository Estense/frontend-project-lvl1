## How to install
1. Node.js 13 or newer is required

2.     $ git clone https://github.com/Estense/frontend-project-lvl1.git

3.     $ npm install make -g

4.     $ make install

5.     $ npm link

6.     Set-ExecutionPolicy -ExecutionPolicy Bypass (only for PowerShell)

7. choose one of these games and type
     $ brain-even
or something listed below
